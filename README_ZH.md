# Moss Lib

[GitHub](https://github.com/Black-Moss/Moss-Lib) / [Nexus Mods](https://www.nexusmods.com/scavprototype/mods/8)

_只是一个[Black_Moss](https://github.com/Black-Moss)的模组前置库。 :)_

这个模组同时包含了[不死Exp](https://github.com/Black-Moss/Undead-Experiment)的内容，所以请不要同时安装！

# 怎么用？

* **在游戏中**运行`undeadmode`命令以启用/禁用不死模式！
* 本地化：详见[ModLocale.cs](Example/ModLocale.cs) & [EN.json](Lang/EN.json) & [ZH.json](Lang/ZH.json)
* 指令：详见[ModCommand.cs](Example/ModCommand.cs)
* 一些工具：详见[Tools.cs](Tools.cs)
