# Moss Lib

[GitHub](https://github.com/Black-Moss/Moss-Lib) / [Nexus Mods](https://www.nexusmods.com/scavprototype/mods/8)

_Just a simple library for [Black_Moss](https://github.com/Black-Moss). :)_

**Note**: This mod also includes content from [Undead Exp](https://github.com/Black-Moss/Undead-Experiment), so please do not install both at the same time!

# How to use?

* Run `undeadmode` command **in-game** to enable/disable undead mode!
* Locale: Look [ModLocale.cs](Example/ModLocale.cs) & [EN.json](Lang/EN.json) & [ZH.json](Lang/ZH.json)
* Command: Look [ModCommand.cs](Example/ModCommand.cs)
* Some tools : Look [Tools.cs](Tools.cs)
